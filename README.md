Extra features you could add to go beyond the tutorial:

1. Quantity controls in cart — increment/decrement reducers, update the slice to store qty per item
2. Toast notifications — show "Added to cart" feedback using react-hot-toast or sonner
3. Cart persistence — redux-persist to survive page refresh via localStorage
4. Category filter — filter products by men's clothing, electronics, etc. on the home page, purely with local state
5. Product detail page — a /product/:id route with full description and rating
6. Search bar — filter products by title in real time
7. Loading skeletons — replace the plain Loading... text with skeleton cards while React Query fetches
